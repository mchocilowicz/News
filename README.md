# News

### Start Backend: 
    - Go to ./backend
    - Run mvn install
    - java -jar target/backend-0.0.1-SNAPSHOT.jar
    
### Start Frontned:
    - Got to ./fronend
    - Run npm install
    - Run npm run start  
    
### Url:
    - swagger-ui -> localhost:8080/swagger-ui.html
    - fronend -> localhost:4200
    - backend -> localhost:8080
